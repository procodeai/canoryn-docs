## 1.0.0-beta — June 15, 2026

The first public beta of Canoryn, the local-first AI workspace agent for macOS.

### Added

- **Menu bar assistant** — run automations instantly with a customizable shortcut.
- **Visual Architect** — drag, connect, and customize triggers, intelligence, and native app nodes.
- **App integrations** — control Finder, Mail, Calendar, Apple Music, and more.
- **Local LLM** — fully private inference via Ollama (Llama 3.2, Qwen 2.5), or bring your own cloud API keys.
- **Portable workflows** — export and version-control your automations as standard `.cryn` files.

### Requirements

- macOS 14 or later (Apple Silicon & Intel).

### Notes

This is an early beta — expect rough edges. On first launch macOS shows a one-time
security prompt (the build isn't notarized yet); see the
[download page](https://canoryn.app/download) for the one-step workaround.
Report issues to [support@procodeai.com](mailto:support@procodeai.com).
